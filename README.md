# zet-site

Static personal page for `zet`.

## Local preview

```bash
python -m http.server 8080 -d zet-site
```

Open `http://localhost:8080`.

## GitHub + Render

1. Push this folder as a repository to GitHub.
2. In Render, create a new Blueprint or Static Site from the repo.
3. If using the included `render.yaml`, Render will publish the current directory.
4. If creating manually, set Publish Directory to `.` and leave Build Command empty.

Music uses local `nateplokhode.mp3` and starts after the enter click.
